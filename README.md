# Mulesoft_Movies

## 1.Create database and create "MOVIES" table and with the given attributes.

## 2.'Movies.sql' file contains SQL code for creating table,inserting values and some queries.

## 3.'dbconnect.php' file connects the XAMPP server to our project.

## 4.'index.php' file is used to include database connection in our project.
